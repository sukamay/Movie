# 电影清单项目

## 一、实现功能：

1. 解析提供的`films.json`文件，并将解析后的内容动态生成了html元素，以列表形式展示在网页中
2. 实现了分页功能
3. 实现了电影详情页
4. 实现了电影排行榜（展示根据评分进行排序后的结果
5. 实现了多关键词搜索功能

## 二、页面简介：

### 1. index.html：

展示电影清单（按照films.json里给的顺序）

❗️注意：因为我是用发送XMLHttpRequest的方式打开json文件的，因此要在像WebStorm类似的提供了web服务器的IDE上打开才能读取



### 2. rank.html：

展示按照评分排序过后的电影清单



### 3. detail.html

展示某部电影的详情界面
